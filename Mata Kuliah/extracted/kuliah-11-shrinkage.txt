2. Apa Itu OLS?

Ordinary Least Squares (OLS)

OLS adalah metode regresi linear yang mencari model terbaik dengan cara meminimalkan jumlah kuadrat error
antara nilai aktual dan nilai prediksi.

y^i : hasil panen yang diprediksi model

Permasalahan OLS: Koefisien Bisa Tidak Stabil

Koefisien OLS dapat berubah cukup besar ketika data sedikit berubah. Hal ini menunjukkan bahwa model
memang bisa fit pada data latih, tetapi belum tentu stabil untuk interpretasi dan prediksi baru.

Makna visualisasi

Setiap box menunjukkan variasi koefisien dari banyak sampel ulang. Semakin lebar variasinya, semakin tidak
stabil estimasi koefisien OLS.

Jadi, Mengapa Perlu Shrinkage?

Ide utamanya sederhana                                     Variabilitas koefisien OLS dibandingkan Ridge.
      OLS hanya fokus mengecilkan error.
      OLS tidak menahan koefisien agar tetap kecil.
      Pada data yang kompleks, model bisa menjadi terlalu
      "percaya diri."

Shrinkage hadir untuk
      tetap meminimalkan error prediksi;
      sekaligus menambahkan penalti pada besar koefisien;
      membuat model lebih stabil dan lebih baik dalam
      generalisasi.

4. Ridge Regression

Gagasan utama Ridge

Ridge menggunakan OLS sebagai dasar, lalu menambahkan penalti terhadap kuadrat koefisien. Dengan
demikian, semua koefisien didorong untuk mengecil.

                               n                        p   2  p

                     min          yi - 0 - xij j               +  j2

                     0, i =1                          j =1     j =1

jika  = 0, Ridge sama seperti OLS;
jika  membesar, penalti makin kuat;
semua koefisien mengecil, tetapi umumnya tidak menjadi nol.

5. Lasso Regression

Gagasan utama Lasso

Lasso juga berangkat dari OLS, tetapi penalti yang digunakan adalah jumlah nilai absolut koefisien. Penalti ini
membuat sebagian koefisien dapat menjadi tepat nol.

                               n                        p   2  p

                     min          yi - 0 - xij j               +  |j |

                     0, i =1                          j =1     j =1

beberapa koefisien menyusut kuat sampai nol;
model menjadi lebih ringkas;
Lasso dapat melakukan seleksi fitur otomatis.

Efek Seleksi Fitur pada Lasso

Interpretasi

Semakin besar penalti, semakin banyak fitur yang dieliminasi dari model. Karena itu, Lasso sangat berguna saat
kita juga ingin menyederhanakan model, bukan hanya meningkatkan kestabilan.

Kapan Menggunakan Ridge dan Kapan Menggunakan Lasso?

Gunakan Ridge jika...                       Gunakan Lasso jika...

      banyak fitur sama-sama penting;             ingin model yang lebih ringkas;
      terdapat multikolinearitas;                 ingin seleksi fitur otomatis;
      fokus utamanya adalah kestabilan dan        diduga hanya sebagian kecil fitur yang dominan.
      akurasi prediksi.

Kalimat penutup

Pada masalah prediksi hasil panen, Ridge membantu ketika banyak variabel sama-sama relevan, sedangkan Lasso
membantu ketika kita juga ingin mengetahui fitur mana yang benar-benar penting.
